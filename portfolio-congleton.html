Project Overview

Modern Functionality Meets Style

This project involved converting an existing conservatory into a fully insulated, year-round living space, paired with a complete kitchen redesign. The result is a seamless open-plan area that blends the existing home with modern functionality and style.

The conservatory conversion included new insulated roofing, underfloor heating, and full integration with the kitchen to create a light-filled, comfortable living space that can be enjoyed all year round.
